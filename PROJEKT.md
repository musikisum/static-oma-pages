# Musikspiele-Sammlung

Statische HTML-Musikspiele, gehostet auf GitHub Pages.

## Regeln

- Jedes Spiel: eigener Unterordner
- Dateien pro Spiel: `index.html`, `style.css`, optional `script.js`
- Gesamtgröße pro Spiel: < 1 MB
- Keine externen Abhängigkeiten (alles inline oder im Ordner)
- Kein Build-Schritt, kein Framework

## Spiele

| Ordner | Beschreibung | Status |
|--------|-------------|--------|
| `toene-im-bass-und-violinschluessel/` | Noten im Bass- und Violinschlüssel erkennen | fertig |

## Struktur

```
static-oma-pages/
├── PROJEKT.md
└── toene-im-bass-und-violinschluessel/
    ├── PROJEKT.md
    ├── index.html
    ├── style.css
    └── script.js
```

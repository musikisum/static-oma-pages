# Musikspiele / Music Games

Interaktive Lernspiele für Musiktheorie — statische HTML-Seiten, eingebettet in die [Open Music Academy](https://openmusic.academy).

Interactive music theory learning games — static HTML pages, embedded in the [Open Music Academy](https://openmusic.academy).

---

## Spiele / Games

### [Töne im Bass- und Violinschlüssel](toene-im-bass-und-violinschluessel/)

**DE:** Eine Note erscheint auf dem Notensystem. Klicke die richtige Taste auf der Klaviertastatur — sofortiges Feedback, Statistik und echter Klavierklang (Salamander Grand Piano).

**EN:** A note appears on the staff. Click the correct key on the piano keyboard — instant feedback, statistics, and real piano sound (Salamander Grand Piano).

Tonumfang / Range: C2–C6 · Violin- und Bassschlüssel / Treble and bass clef

---

## Technik / Technical

- Reines HTML/CSS/JS — kein Framework, kein Build-Schritt / Pure HTML/CSS/JS — no framework, no build step
- Jedes Spiel < 1 MB / Each game < 1 MB
- Web Audio API (Salamander Grand Piano via [Tone.js](https://tonejs.github.io/))
- Gehostet auf / Hosted on: GitHub Pages
